                               # Projeto de Análise de Desempenho e Tendências do Sistema Fotovoltaico no Brasil

Este projeto tem como objetivo analisar as tendências de crescimento e queda da energia solar no Brasil, considerando o período de 2009 a 2025. A análise busca identificar padrões de sazonalidade, variações anuais e fatores que influenciam o desempenho do sistema fotovoltaico no país. Serão utilizadas técnicas de visualização de dados e estatística para compreender o comportamento da geração de energia solar ao longo dos anos e apoiar tomadas de decisão para o setor. os dados utilizados serão obtidos através da API da ANEEL, que fornece informações detalhadas sobre a geração de energia solar no Brasil.
